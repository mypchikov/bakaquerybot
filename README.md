## Inline Gelbooru Bot

The bot handles inline queries and returns images from Gelbooru.

Setup:

```bash
BOT_TOKEN=your_token
GELBOORU_USER_ID=your_user_id
GELBOORU_API_KEY=your_api_key
GELBOORU_API_URL=https://gelbooru.com/index.php
INLINE_RESULTS_LIMIT=20
```

`GELBOORU_USER_ID` and `GELBOORU_API_KEY` are optional in code, but in practice Gelbooru may return `401` without them.

In `@BotFather` enable:

```text
/setinline
```

Usage:

```text
@your_bot touhou wings
```
